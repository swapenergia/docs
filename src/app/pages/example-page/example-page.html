Ejemplos de Uso

Descubre cómo usar IAWOT Method en la práctica

1

Fase de Planificación

Comienza con el agente Analyst para crear un brief del proyecto:

# En la interfaz web
*analyst
"Necesito crear una aplicación de e-commerce para pequeñas empresas"
# Luego cambia al PM para el PRD
*pm
2

Transición al IDE

Una vez completados PRD y Arquitectura, fragmenta los documentos:

# En tu IDE con IAWOT instalado
npm run bmad:shard
# Esto creará historias de desarrollo detalladas
3

Desarrollo con Contexto

Los agentes Dev trabajan con historias que contienen contexto completo:

  • Qué construir exactamente
  • Cómo implementarlo
  • Orientación arquitectónica
  • Contexto del proyecto completo
