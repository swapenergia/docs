¿Qué es IAWOT Method?

Un framework universal de agentes IA que transforma cualquier dominio con especialización en inteligencia artificial.

Requisitos del Sistema

Node.js versión 20 o superior
Windows 10/11, macOS 10.15+, Linux Ubuntu 18.04+
Acceso a servicios de IA (Gemini, ChatGPT, etc.)
Conexión a Internet estable
IDE compatible (VS Code recomendado)

Beneficios Clave

Eliminación de Inconsistencias

Sin pérdida de contexto entre planificación y desarrollo

Framework Universal

Funciona en cualquier dominio con lenguaje natural

Expansion Packs

Extiende a escritura, estrategia, wellness y más

Flujo de Trabajo IAWOT

Fase 1 - Planificación (Web UI): Crear PRD y documentos de arquitectura

Fase 2 - Desarrollo (IDE): Los agentes SM, Dev y QA colaboran a través de archivos de historias
