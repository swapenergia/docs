Instalación

Comienza con IAWOT Method en minutos

Prerequisitos

  • Windows con WSL Ubuntu 24.04
  • Node.js v20 o superior
  • Claude Code con licencia MAX
  • IDE ( VSCode, Cursor...)

Si ya tienes los prerequisitos sino sigue el paso a paso:

npx bmad-method install

O clonar el repositorio y usar el instalador

git clone https://github.com/bmadcode/bmad-method.git npm run install:bmad
wsl --install -d Ubuntu-24.04 curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash nvm install 22 claude

Sigue las instrucciones para autenticar tu cuenta, te pedirá autenticarte mediante URL

npx bmad-method install

O clonar el repositorio y usar el asistente

git clone https://github.com/bmadcode/bmad-method.git cd bmad-method/ npm install npm run install:bmad

¿Cómo funciona el Asistente?

El asistente te preguntará dónde quieres crear tu proyecto:

  • Se ejecuta desde donde clonaste el repositorio
  • Puedes usar rutas relativas desde esa ubicación
  • Ejemplo: ../../nuevo_proyecto para crear el proyecto dos niveles arriba
  • Ejemplo: ../mis_proyectos/mi_app para crear en una carpeta específica
