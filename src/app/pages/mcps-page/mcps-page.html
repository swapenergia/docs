Model Context Protocol

Extiende las capacidades de Claude con herramientas especializadas

Sequential Thinking

Mejora el razonamiento paso a paso para análisis más profundos.

npm install -g @modelcontextprotocol/server-sequential-thinking

Context7

Acceso a base de conocimientos y contexto extendido.

claude mcp add --transport http context7 https://mcp.context7.com/mcp

Puppeteer

Automatización web y scraping inteligente.

npm install -g @modelcontextprotocol/server-puppeteer

Magic UI Design

Generación automática de interfaces modernas.

npm install -g @magicuidesign/mcp@latest

Script de Instalación Completa

Instala todos los MCPs de una vez:

# Instalar todos los MCPs
npm install -g @modelcontextprotocol/server-sequential-thinking
npm install -g @modelcontextprotocol/server-puppeteer
npm install -g @magicuidesign/mcp@latest

# Añadir a Claude
claude mcp add-json sequential-thinking '{"command": "npx", "args": ["-y", "@modelcontextprotocol/server-sequential-thinking"]}'
claude mcp add --transport http context7 https://mcp.context7.com/mcp
claude mcp add-json puppeteer '{"command": "npx", "args": ["-y", "@modelcontextprotocol/server-puppeteer"]}'
claude mcp add-json magic '{"command": "npx", "args": ["-y", "@magicuidesign/mcp@latest"]}'

Notas Importantes

  • • Los MCPs requieren Node.js 18+ instalado
  • • Reinicia Claude Desktop después de instalar
  • • Verifica con: claude mcp list
