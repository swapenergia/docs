Model Context Protocol

Extiende las capacidades de Claude con herramientas especializadas

npm install -g @modelcontextprotocol/server-sequential-thinking claude mcp add --transport http context7 https://mcp.context7.com/mcp npm install -g @modelcontextprotocol/server-puppeteer npm install -g @magicuidesign/mcp@latest
# Instalar todos los MCPs npm install -g @modelcontextprotocol/server-sequential-thinking npm install -g @modelcontextprotocol/server-puppeteer npm install -g @magicuidesign/mcp@latest # Añadir a Claude claude mcp add-json sequential-thinking '{"command": "npx", "args": ["-y", "@modelcontextprotocol/server-sequential-thinking"]}' claude mcp add-json puppeteer '{"command": "npx", "args": ["-y", "@modelcontextprotocol/server-puppeteer"]}' claude mcp add --transport http context7 https://mcp.context7.com/mcp claude mcp add-json magic '{"command": "npx", "args": ["-y", "@magicuidesign/mcp@latest"]}'

Notas Importantes

  • • Los MCPs requieren Node.js 18+ instalado
  • • Reinicia Claude Desktop después de instalar
  • • Verifica con: claude mcp list
